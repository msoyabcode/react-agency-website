# React Agency Website

A modern, responsive agency website built using **React.js**. This project demonstrates the use of React components, React Router, and modern CSS styling to create a professional website layout suitable for agencies, freelancers, or small businesses.

## Features

- Fully responsive design (desktop, tablet, mobile)
- Multi-section layout: Home, About, Services, Portfolio, Contact
- Smooth scrolling navigation
- Modern UI/UX design
- Reusable React components
- Easy to customize and extend

## Technologies Used

- **React.js** – Frontend library for building UI
- **JavaScript (ES6+)** – Logic and interactivity
- **HTML5 & CSS3** – Structure and styling
- **React Router** – For navigation (if used)
- **Optional**: Tailwind CSS / Bootstrap (if you used any)
